Getting Started & Human Verification

The MB-820 Quiz is a browser-based study tool designed to help you prepare for the Microsoft Dynamics 365 Business Central Functional Consultant (MB-820) certification exam. No installation is required — simply open index.html in your browser.

Figure 1.1 — Human Verification Screen
A simple math challenge protects the quiz from automated access.

What is the verification?

When you first open the quiz, a Human Verification card appears over a blurred background. This is a lightweight bot-protection mechanism that ensures real users are accessing the content.

1
A random arithmetic question is shown (e.g., "What is 13 − 7?")
2
Type the numeric answer in the input field
3
Click Enter (or press Enter) to proceed
4
A new question is generated if you answer incorrectly — no lockout
💡 Tip: The verification only runs once per session. Refreshing the page will show it again.

Main Dashboard Overview

After passing verification, you land on the Main Dashboard — your hub for all quiz activities. It is organized into distinct sections that scroll vertically.

Figure 2.1 — Main Dashboard (top portion)
The dashboard shows your preparation progress, in-progress sessions, Quick Practice, and all available quiz sets.

Dashboard Sections

🎯 Preparation Progress Bar

Tracks how many quizzes and case studies you have completed with ≥70%. Fills to 100% when everything is passed.

⏸ In-Progress Banner

Shown when you have paused or navigated away mid-quiz. Reminds you to continue where you left off.

⚡ Quick Practice

One question at a time, no timer, instant feedback. Draws from all 277 questions and prioritizes ones you previously got wrong.

📚 Practice Quizzes

Four structured quiz sets (Beginner → Official) each with a 120-minute timer. Can be combined with a Case Study.

🎲 Random Practice

67 randomly selected questions from all sets. Results don't count towards preparation progress.

📋 Case Studies

3 standalone scenario-based test cases (Alpine, Contoso, Fabrikam). Can also be run after a Practice Quiz for a combined score.

💾 Export / Import

Transfer all your progress between devices by exporting to a JSON file and importing on another browser.

🗑️ Reset All Progress

Wipes all saved data including scores, history, and timers. Individual quiz/case reset buttons are also available per card.

Quick Practice

Quick Practice is the fastest way to squeeze in focused study. It sits at the top of the dashboard and serves one question at a time with no countdown pressure.

Figure 3.1 — Quick Practice card
Figure 3.2 — Streak milestone overlay

How it works

  • Questions are drawn from all 277 questions across every quiz set and case study.
  • Questions you previously answered incorrectly appear first, then unseen ones, then correctly-answered ones.
  • Clicking Submit Answer immediately reveals whether you were right or wrong, along with the explanation.
  • Use Skip ⏭ to jump over a question you're not ready for.
  • Use 📋 Copy to copy the question details to your clipboard to report an issue via Teams.

🔥 Streak System

Every consecutive correct answer increases your streak counter. Milestone achievements unlock a celebration overlay:

🔥 5 in a row

KILLING SPREE! — Keep the momentum going.

💥 10 in a row

RAMPAGE! — You're on fire!

⚡ 15 in a row

DOMINATING! — Impressive knowledge.

🌪️ 20 in a row

UNSTOPPABLE! — You know this material cold.

⚠️ Note: If you get a question wrong, your streak resets to 0 and the question pool resets so all questions become available again — giving you a fresh chance to build your streak.

Practice Quiz Sets

There are four structured quiz sets designed to progressively challenge you, plus a random practice option. Each has a 120-minute countdown timer mirroring the actual exam conditions.

🟢 Beginner Quiz

66 foundational questions covering core Business Central concepts and terminology. Perfect starting point for newcomers.

🟡 Intermediate Quiz

67 questions bridging foundational knowledge and advanced Business Central topics. For those with some BC experience.

🔴 Proficient Quiz

67 scenario-based questions balanced across the full MB-820 exam: AL development, events, testing, permissions, API pages, XMLports, telemetry, and deployment.

📋 Official Questions

77 official MB-820 exam questions covering deployment architecture, AL development, permissions, XMLports, queries, HTTP, reports, telemetry, and more.

🎲 Random Practice Quiz

67 randomly selected questions drawn from all available sets. Great for mixed revision. Note: results do not count towards the preparation progress bar.

Card Actions

Each quiz card offers the following buttons depending on its state:

  • Start Quiz — Begin a fresh quiz (prompts mode selection)
  • Resume — Continue an in-progress session exactly where you left off
  • New Quiz — Discard saved progress and start over (confirmation required)
  • 🔍 Review Results — Re-read a completed quiz's answers without re-taking it
  • 🔄 Reset — Delete all progress for this specific quiz only

📋 Attaching a Case Study

Each Practice Quiz card has a Case Study (optional) dropdown. Selecting a case study (Alpine, Contoso, or Fabrikam) runs the quiz and the case study back-to-back, with a combined weighted score (70% quiz + 30% case study) shown on the results screen.

✅ Pass threshold: A score of 70% or higher is required to mark a quiz or case study as completed in the preparation progress bar — matching the real MB-820 pass mark.

Answering Questions

Questions come in two types, clearly indicated by a colored badge at the top of each question.

Figure 6.1 — Multiple choice question
Purple badge: "Multiple Choice — select all that apply". Use checkboxes.
Figure 6.2 — Single choice question (wrong answer)
Blue badge: "Single Choice". Use radio buttons. Red = your wrong pick, Green = correct answer.

Question Types

Single Choice

Select exactly one answer using radio buttons. The Submit Answer button becomes active immediately upon selection.

Multiple Choice

Select all correct answers using checkboxes. The question text specifies the exact count (e.g. "Select TWO", "Select ALL that apply").

Figure 6.3 — Correct answer feedback (Practice Mode)
Green highlights show which answers were correct. A green explanation box beneath confirms the right answer and explains why.

Answer Feedback Colors

🟩 Green border

This is the correct answer (shown regardless of what you selected).

🟥 Red border

You selected this answer but it is incorrect.

📋 Copy Issue Button

Every question has a small 📋 Copy Issue button. Clicking it copies the full question details (text, choices, correct answer) to your clipboard in a formatted template — ready to paste into a Teams chat to report a bug or typo.

💡 Partial credit: No partial credit is awarded for multiple-choice questions — you must select all correct options and no incorrect ones to receive full credit.

Timer & Auto-Pause

Every Practice Quiz runs with a 120-minute countdown timer matching the actual MB-820 exam duration. The timer is displayed prominently in the header bar throughout the quiz.

Timer Color States

🟢 > 30 min — Plenty of time
🟡 10–30 min — Getting close
🔴 < 10 min — Hurry up!

When the timer reaches zero, it displays "⏱ 00:00 — Time's up!" but does not forcibly submit your quiz.

⏸ Auto-Pause

Figure 7.1 — Auto-Pause overlay
The quiz automatically pauses after 2 minutes of inactivity.

If you stop interacting with the page for 2 minutes, the quiz automatically pauses and displays this overlay. Your timer also stops — so you won't lose time while away.

▶
Continue Quiz — Resumes from exactly where you paused, timer restarts
🏠
Main Page — Saves progress and returns to the dashboard; you can Resume later
💡 Tip: Activity that resets the idle timer includes mouse movement, clicks, key presses, scrolling, and touch events.

Progress Auto-Save

The quiz automatically saves your position and remaining time every 10 seconds. If your browser crashes or you accidentally close the tab, you can return and click Resume from the quiz card to pick up where you left off.

Test Mode & Peek Score

In Test Mode, no answer feedback is given during the quiz — just like the real exam. To keep you motivated (and a little nervous), there is a Peek Score feature.

Figure 8.1 — Peek Score overlay
Peek at your partial score mid-quiz in Test Mode — complete with a taunting message!

How to Peek

In Test Mode, a Peek Score button appears next to the Submit button. Click it at any time to open the Peek overlay and see your current running percentage.

📊

😏

Results & Answer Review

After answering all questions (or after time runs out), the Score Summary screen is displayed.

Figure 9.1 — Score Summary screen
The results screen shows your overall percentage, pass/fail status, time used, and a breakdown of correct vs wrong answers.

What the results screen shows

  • 🎉 / 😞 Overall emoji and pass/fail message based on your score
  • Percentage score and fraction (e.g., 52 / 66 correct)
  • ✅ PASS (≥ 70%) or ⚠️ FAIL (< 70%)
  • Time used, quiz set name, and mode (Practice / Test)
  • Count of correct and wrong answers
  • List of questions you got wrong, showing your answer vs. the correct answer
  • Confetti animation 🎉 for a passing score
🏠 Main Page

Returns to the dashboard. The quiz card now shows your last attempt badge (✅ or ⚠️) with the percentage.

🔍 Review Answers

Re-opens the quiz in read-only review mode so you can read through every question, your answer, and the explanation.

✅ After passing: The quiz card on the dashboard shows a green "Last attempt: 78%" badge and a 🔍 Review Results button to re-read your answers at any time. Your score is saved permanently.

Select a case study from the Case Study (optional) dropdown on a Practice Quiz card before starting. The quiz runs first (with timer), then the case study follows. A combined weighted score is shown.

Progress Tracking & History

The quiz tracks your study progress in several ways, all stored locally in your browser via localStorage.

🎯 Preparation Progress Bar

The bar at the top of the dashboard shows how many of the 7 items (4 quizzes + 3 case studies) you have completed with ≥ 70%:

🎯 MB-820 Preparation Progress 3 / 7 (43%)

Complete every quiz and case study with ≥70% to reach 100% preparation.

  • Bar turns blue → green as you approach 100%
  • A 🎉 full bar indicates you are fully prepared
  • Study time (total time spent in active quizzes) is shown below the bar

📋 Attempt History

Scrolling to the bottom of the dashboard reveals your Previous Attempts section — a chronological list of every quiz and case study you have completed, with:

  • Date and time of the attempt
  • Quiz / case study name and type (standalone, combined, or case-only)
  • Score percentage, colour-coded: ✅ Pass ≥70% ⚠️ Marginal 60-70% ✗ Fail <60%

Up to 50 attempts are retained. Use the 🗑️ Clear History button to remove the history log while keeping your current best scores.

💡 Tip: Progress is stored per-browser. If you switch browsers or clear site data, your progress will be lost. Use the Export Progress feature (Section 12) to keep a safe backup.

Export & Import Progress

The Export / Import feature at the bottom of the dashboard allows you to back up and transfer your study progress between devices or browsers.

⬆️ Export Progress

Downloads a .json file (named mb820-progress-YYYY-MM-DD.json) containing all your quiz results, timers, history, and streak data. Store it safely or transfer it to another device.

⬇️ Import Progress

Opens a file picker. Select a previously exported .json file. A confirmation prompt warns you that existing data will be overwritten. The page reloads with your restored progress.

  1. 1
    On your current device, scroll to the bottom of the dashboard and click ⬆️ Export Progress.
  2. 2
    A file named mb820-progress-YYYY-MM-DD.json is downloaded to your computer.
  3. 3
    Transfer the file to your new device (e.g., via email, USB, OneDrive).
  4. 4
    Open the quiz on the new device, scroll to the bottom, and click ⬇️ Import Progress.
  5. 5
    Select the exported JSON file. Confirm the overwrite prompt and your progress is restored.
⚠️ Warning: Importing will overwrite all existing progress on the target device. Make sure to export from the target device first if you need to preserve anything there.

Tips & Best Practices

📅 Recommended Study Path

  1. 1
    Start with Beginner Quiz in Practice Mode — Learn the terminology and foundational concepts before attempting harder questions.
  2. 2
    Use Quick Practice daily — Even 5–10 minutes per day reinforces weak areas since failed questions surface first.
  3. 3
    Progress through Intermediate → Proficient → Official — Each level adds new exam topic coverage.
  4. 4
    Tackle Case Studies standalone first — Get familiar with the scenario context before combining them with a timed quiz.
  5. 5
    Do a final full combined run in Test Mode — Simulate the actual exam by picking Proficient or Official + a Case Study, switching to Test Mode.

🎯 Pro Tips

  • ⏱Don't rely on the timer too much — the 120-minute limit is generous. Focus on accuracy, not speed.
  • 🔍Always use Review Results after failing. Read every explanation, especially for wrong answers.
  • 🔄Reset a quiz and retake it a few days later to test long-term retention, not just short-term memory.
  • 📋Case Study scenario text is shown as a header during each question — you don't need to memorise the scenario.
  • 💾Export your progress before clearing browser data or switching to a private browsing window.
  • 🔥Chase the Quick Practice streak! Reaching 10+ in a row is a good confidence indicator before exam day.
  • 📱The quiz is mobile-friendly — you can study on your phone during commutes (Quick Practice works great on mobile).

🛠️ Troubleshooting

Progress not saving

Check that your browser allows localStorage. Incognito/Private mode may block it. Use a regular browser window.

Found a wrong answer?

Click the 📋 Copy Issue button on any question to copy a report template, then send it via Teams chat to the quiz maintainer.

Quiz won't load

Make sure all .js files are present in the same folder as index.html. The quiz requires no internet connection.

Want to start fresh

Use 🗑️ Reset All Progress at the bottom of the dashboard to clear everything, or reset individual quizzes using their 🔄 Reset buttons.
